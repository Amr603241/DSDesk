========================================================================
   DSDesk Professional Remote Desktop - User Guide
========================================================================

Welcome to the professional edition of DSDesk. This package is designed 
for high-performance, secure remote control.

QUICK START:
1. Extract the contents of this ZIP file to a folder.
2. Double-click "DSDesk.exe" to start.
3. To enable full control (including Task Manager), right-click 
   "DSDesk.exe" and select "Run as Administrator".

SECURITY NOTE (SmartScreen):
As a new, high-performance software, Windows may occasionally show 
a "Windows protected your PC" message. This is normal for unsigned 
professional tools. 
To proceed:
- Click "More Info"
- Click "Run anyway"

TECHNICAL SUPPORT:
Designed for low-latency, 1:1 precision remote desktop control.
